Descripción detallada

Define un conjunto (set) en lógica difusa. Un conjunto define, mediante valores (TFuzzyVal), los diferentes estados de una propiedad medible. Un ejemplo de conjunto podría ser la temperatura de una caldera, y algunos de sus valores serían: Caliente, Frío, Templado...
